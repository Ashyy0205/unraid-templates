# unraid-templates

Unraid Community Applications templates by [Ashyy0205](https://github.com/Ashyy0205).

## Templates

### SoundScout

A self-hosted music discovery and acquisition app for Plex.

- Uses Last.fm listening history to build personalised recommendations
- Import Spotify playlists and albums
- Downloads missing music into your Plex library
- Keeps Plex playlists in sync via a Web UI
- Per-user Last.fm accounts, discovery schedules, and artist monitoring

**Docker image:** `ghcr.io/ashyy0205/soundscout:main`  
**Source:** [github.com/Ashyy0205/SoundScout](https://github.com/Ashyy0205/SoundScout)

## Usage

These templates are consumed automatically by Unraid Community Applications every ~2 hours.  
To install manually, add this repository URL in the CA Apps settings:

```
https://github.com/Ashyy0205/unraid-templates
```
